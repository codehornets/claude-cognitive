# Configure GitHub Repository

Your repo is live: https://github.com/GMaN1911/claude-cognitive

**Do these now (5 minutes):**

---

## 1. Add Topics/Tags

1. Go to: https://github.com/GMaN1911/claude-cognitive
2. Click ⚙️ (gear icon) next to "About" on right side
3. Add these topics (one at a time):
   - `claude-code`
   - `claude-ai`
   - `context-management`
   - `token-optimization`
   - `developer-tools`
   - `productivity`
4. Click "Save changes"

**Why:** Topics help people discover your repo via GitHub search.

---

## 2. Enable Discussions

1. Go to: https://github.com/GMaN1911/claude-cognitive/settings
2. Scroll to "Features" section
3. Check ✅ **Discussions**
4. Click "Set up discussions"
5. GitHub will create default categories (General, Q&A, etc.)

**Optional - Create Custom Categories:**
1. Go to Discussions tab
2. Click "New discussion"
3. You'll see categories on the right
4. For now, default categories are fine

**Why:** Discussions give users a place to ask questions without opening Issues (which are for bugs).

---

## 3. Add Repository Description

1. Go to: https://github.com/GMaN1911/claude-cognitive
2. Click ⚙️ (gear icon) next to "About"
3. **Description:** `Working memory for Claude Code - persistent context and multi-instance coordination`
4. **Website:** (leave blank or add https://mirrorethic.com)
5. Click "Save changes"

---

## 4. Enable Issues (Should Already Be On)

1. Go to: https://github.com/GMaN1911/claude-cognitive/settings
2. Scroll to "Features"
3. Make sure ✅ **Issues** is checked

**Why:** Users will report bugs and ask setup questions here.

---

## 5. Pin the SETUP.md

1. Go to: https://github.com/GMaN1911/claude-cognitive
2. Click on `SETUP.md`

4. Users will naturally find this

Already done! README links to SETUP.md multiple times.

## 6. Watch Your Own Repo

1. Go to: https://github.com/GMaN1911/claude-cognitive
2. Click "Watch" dropdown (top right)
3. Select "All Activity"

**Why:** You'll get email notifications for Issues, Discussions, Stars, etc.

1. Go to: https://github.com/GMaN1911/claude-cognitive/releases
2. Click "Create a new release"
3. **Tag:** `v1.0.0`
4. **Title:** `v1.0.0 - Initial Release`
5. **Description:** Copy from CHANGELOG.md (v1.0.0 section)
6. Click "Publish release"

**Why:** Releases make it easy for users to download specific versions.
